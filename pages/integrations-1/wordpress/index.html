We support WordPress too! In order to use HAX though you'll want to replace the Gutenberg editor. The following steps will help you get it up and running.

  1. Get the Classic Editor . This plugin helps restore WordPress back to what you wanted
  2. Get the Web component plugin , unzip and rename to webcomponents
  3. Get the HAX plugin , unzip and rename to hax
  4. Put both of these plugins in your wp-content/plugins/  directory
  5. Go into your site's plugins area and enable both wp-admin/plugins.php
  6. Break free of platform specific assets and content
